veliwrap

Note : Le wrap est plus simple à effectuer sur ordinateur, mais il est tout de même possible de le faire sur mobile.

Étape 1 : connectez-vous au site Vélib.

Étape 2 : cliquez sur ce lien et téléchargez la page (Ctrl+S ou en faisant clic droit). cliquez sur le bouton de téléchargement comme sur l'image ci-dessous :

Étape 3 : uploadez le fichier en cliquant sur le bouton ci-dessous

Le fichier contient les informations sur tous vos trajets en vélib associé à votre compte. Aucune donnée n'est enregistrée ni même envoyée à mes serveurs, tout se fait en local, dans votre navigateur. Le code source du projet est disponible sur GitHub. Ce site n'est pas affilié à l'entreprise Vélib et est un projet indépendant.
